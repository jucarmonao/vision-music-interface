# Vision Music Interface

Interactive system that connects **computer vision, music, and inclusive education**.

This project explores the intersection between **technology, art, and human interaction**, transforming visual input into real-time musical responses.

Using computer vision, users can interact with musical instruments such as **guitar and trumpet**, triggering notes and chords through hand movements detected by the camera.

Beyond a technological prototype, the project was conceived as an **educational and inclusive platform**, designed to demonstrate how **STEM environments can be accessible to diverse communities**, including children, elderly people, and individuals with motor difficulties.

# How it Works

The system uses **computer vision** to detect hand gestures and translate them into musical responses in real time.

Each finger movement activates a different note or chord depending on the selected instrument and interaction mode.

This approach transforms the camera into a **contactless musical interface**, allowing intuitive interaction without traditional instruments.

---

# Interaction Modes

**1 → Trumpet Mode**

The system triggers trumpet notes through finger interaction.

**2 → Guitar Mode**

The system activates guitar chords and harmonic structures.

**0 → Individual Mode**

Single-user interaction with one instrument.

**9 → Paired Interaction**

Two users interact simultaneously with the same instrument.

**8 → Hybrid Paired Mode**

Two users interact simultaneously with mixed instruments (guitar + trumpet).

This configuration allows musical experiences that range from **individual exploration to collaborative performance**.

# Social and Inclusive Impact

One of the main motivations behind this project is **accessibility and inclusion**.

The system enables musical interaction without requiring traditional instrumental training or fine motor skills.

This opens possibilities for:

### Children in underserved educational contexts

The platform can act as an **entry point to STEM learning** in communities with limited access to technology or formal music education.

Through playful interaction, children can explore technology while expressing themselves creatively.

### Elderly people

The system can be used in **senior centers and nursing homes** as a tool for:

- cognitive stimulation
- emotional expression
- playful interaction

Music interaction can generate meaningful engagement even for people without previous musical experience.

### People with motor difficulties

Because the system relies on **large, simple hand movements**, it can be adapted for individuals with:

- reduced mobility
- limited motor precision
- rehabilitation processes

This makes the interface a potential **inclusive creative tool**, where music becomes accessible through gesture rather than traditional instrument technique.

# Funcionamiento

El sistema utiliza **visión por computadora** para detectar gestos de las manos y transformarlos en respuestas musicales en tiempo real.

Cada movimiento de los dedos activa diferentes notas o acordes dependiendo del instrumento seleccionado y del modo de interacción.

Esto convierte la cámara en una **interfaz musical sin contacto**, permitiendo una interacción intuitiva sin necesidad de instrumentos tradicionales.

---

# Modos de interacción

**1 → Modo trompeta**

Activa notas de trompeta mediante interacción visual.

**2 → Modo guitarra**

Activa acordes y estructuras armónicas de guitarra.

**0 → Modo individual**

Interacción de un solo usuario con un instrumento.

**9 → Interacción en parejas**

Dos usuarios interactúan simultáneamente con el mismo instrumento.

**8 → Modo híbrido en parejas**

Dos usuarios interactúan simultáneamente con instrumentos distintos (guitarra + trompeta).

Esto permite experiencias musicales que van desde la **exploración individual hasta la interpretación colaborativa**.
