# Vision Music Interface

Interactive musical interface based on computer vision.

This project allows users to generate musical notes using hand gestures detected through a webcam. The system uses hand landmark detection to interpret finger positions and trigger different sounds.

The goal of the project is to explore new ways of interacting with music through technology and to create accessible musical experiences using only a computer and a camera.

## Technologies used

- Python
- OpenCV
- MediaPipe
- Pygame

## Features

- Hand detection using computer vision
- Gesture recognition using hand landmarks
- Real-time sound triggering
- Multiple instrument modes
- Support for one or two players

## Project motivation

This project explores the intersection between technology, music and education. It was developed as part of an initiative to demonstrate how digital tools can enable musical interaction in contexts where physical instruments may not be available.

The system aims to support educational and creative environments such as STEM labs and technology learning spaces.

## How to run the project

1. Install dependencies

```
pip install -r requirements.txt
```

2. Run the program

```
python main.py
```

3. Use your hands in front of the camera to trigger musical notes.
